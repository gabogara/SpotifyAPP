Project Overview

Explore artists, popular songs, genres and albums using the Spotify API.

How It Works

This project interacts with the Spotify API to retrieve and display artist-related information through four key API calls:

Search for an Artist

Enter an artist's name in the search field to fetch and display relevant information about them.

Popular Songs

Clicking on an artist redirects you to a dedicated page showcasing their most popular songs. This data is retrieved using the artist's ID, passed via the URL.

Albums

A dedicated page displays all the albums of the selected artist, fetched using their ID from the API.

Musical Genres

In the main page we can see the result of the call to the API to obtain the musical genres.
